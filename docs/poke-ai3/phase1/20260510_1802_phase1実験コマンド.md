# Phase 1 実験コマンド

まずはこれで短めに動作確認できます。

```bash
cd /home/dochy/pokemon_ai_proj/poke-ai3-python
uv run phase1-loop --num-games 2 --chunk-threshold 4 --trajectories-threshold 2 --max-epochs 3
```

少し実験らしく回すなら:

```bash
cd /home/dochy/pokemon_ai_proj/poke-ai3-python
uv run phase1-loop --num-games 16 --chunk-threshold 32 --trajectories-threshold 16 --max-epochs 100
```

出力の `power80_rate` が上がっていくかを見るのが最初の確認ポイントです。

デフォルトではcheckpointを読み書きせず、新規モデルとして学習します。
checkpointから再開し、同じファイルへ保存したい場合は`--checkpoint-path`を指定します。

```text
/home/dochy/pokemon_ai_proj/data/poke-ai3/checkpoints/phase1_latest.pt
```

checkpointを使って実行する例:

```bash
cd /home/dochy/pokemon_ai_proj/poke-ai3-python
uv run phase1-loop --num-games 16 --chunk-threshold 32 --trajectories-threshold 16 --max-epochs 100 --checkpoint-path /home/dochy/pokemon_ai_proj/data/poke-ai3/checkpoints/phase1_latest.pt
```

checkpointを消したい場合:

```bash
rm /home/dochy/pokemon_ai_proj/data/poke-ai3/checkpoints/phase1_latest.pt
```
