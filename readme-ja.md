# ポケモンにおけるナッシュ均衡の研究

*Read this in [English](readme.md).*

このリポジトリは、ポケモンAIを作り、ナッシュ均衡がどこにあるのかを調べるプロジェクトのリポジトリである。

## 極端な有利対面によるサイクル戦　ヒスイヌメルゴン vs パルシェン (Stage 3b)

[ソースコード](https://github.com/dochy-ksti/pokemon-nash-study)

目的:極端な"有利/不利"対面を作り、"交代/攻撃"一択になるのかを調べる。

概要:とくぼう特化のヒスイヌメルゴンと、ぼうぎょ特化のパルシェンにチームを組ませ、持ち技を工夫することで完全な"有利/不利"対面を作ってAIに学習させてみた。

こうげきととくこうの数値を一致させ、さらにぼうぎょととくぼう以外の実数値は全て一致、ぼうぎょととくぼうだけ反転させたポケモン二匹のチームを作った。わざはパルシェンに対して「でんげきは(特殊:でんき:威力60)」、ヒスイヌメルゴンに対しては「じならし(物理:じめん:威力60)」で弱点をつけるが、もう一匹に対しては極端にダメージが低くなるようにし、交代すればダメージを抑えられる状況にした。持っているわざがチームで反転するようにし、お互いに「不利対面から交代すれば有利対面になる」という、サイクル戦が起こりやすいチーム編成にしてAIに学習させ、AIの戦略がどうなるかを調べた。もちものはお互いに隠密マントを持たせ、わざの追加効果を封じた。

[▶ 実際に対戦できるページ](https://pokemon-nash-study.pages.dev/battle-3b.html)

結果：有利対面に見えても、かなり複雑な読み合いが生じてしまった。

特に難しかったのは、実在するわざの制限により、完全な対称関係を作ることが出来ず、じならしがパルシェンに等倍、でんげきはがヒスイヌメルゴンに半減になってしまったことで、「有利対面でもでんげきはを撃たずに釣り交換し、じならしをうつ」という戦略が非常に効果的になってしまった。

## 真・極端な有利対面によるサイクル戦　ヌメルゴン vs パルシェン (Stage 3c)

[ソースコード](https://github.com/dochy-ksti/pokemon-nash-study)

目的:真に極端な"有利/不利"対面を作り、"交代/攻撃"一択になるのかを調べる。

概要:とくぼう特化のヌメルゴンと、ぼうぎょ特化のパルシェンにチームを組ませた。さらに、実在しない「威力60の物理フェアリー技」「威力60の特殊かくとう技」を覚えさせ、対戦させた。

実数値は同じで、タイプ相性関係も完全に対称にしてどうなるか調べた。

対戦ページは**準備中 (Coming soon)**。

結果：だいぶシンプルになったが、やはり複雑な読み合いが生じた。

相手のどちらか一体だけを倒し、交代を封じることで有利になれるので、相手の一匹を集中攻撃するための釣り交換が戦略として有効だった。結果として、有利対面でも攻撃一択にはならず、不利対面でも交代一択にはならなかった。

## 技術メモ

対戦デモは、Rust製バトルシミュレータ (poke-sho-rust) を WebAssembly 化してブラウザ内でターン解決し、AIの手番は全状態を事前計算したポリシーテーブルを引くだけで決まる。したがってサーバもGPUも不要で、静的ホスティング (Cloudflare Pages) のみで動作する。
