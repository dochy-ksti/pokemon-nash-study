# Stage 3c: warmup 50ep vs 400ep 強さ A/B + tokio スレッド数ベンチ

実行完了: 2026-07-05 (JST) / stage 3c・K1b5 レシピ (K=1, block=5, r=0.5)

## 目的

1. K1b5 レシピを stage 3c に適用し、**自己対戦 warmup を 50ep と 400ep** に変えて強さに差が出るか調べる。
2. (ついで) tokio ワーカースレッド数を絞ると CPU 使用率/速度が変わるかを小規模ベンチで確認し、
   最速設定を本番 A/B に採用する。

## フェーズ0: スレッド数ベンチ (速度のみ)

固定仕事量 = 60 追加epoch (warmup 50 + HtH 2ブロック)。funnel を stage 3c・K1b5 レシピで各1本、
`/usr/bin/time -v` で計測。冒頭に捨て 20ep (GPU warm)。

| 設定 | Elapsed | epochs/分 | %CPU |
|---|---:|---:|---:|
| 既定 (32論理スレッド) | 4:05.8 | 14.65 | 587% |
| **taskset -c 0-11 / TOKIO_WORKER_THREADS=12** | **3:52.7** | **15.48** | **422%** |
| taskset -c 0-15 / TOKIO_WORKER_THREADS=16 | 4:00.3 | 14.98 | 477% |

- **taskset12 が最速かつ最省CPU** (既定比 約5%速・CPU 587→422%)。GPU 律速の生成で tokio スレッドを
  減らすと競合が減りコア浪費が止む。差自体は各1本ぶんのノイズ帯だが wall/CPU 両方で最良。
- → 本番 A/B は全アーム `taskset -c 0-11 env TOKIO_WORKER_THREADS=12` で実行。

## フェーズ1: 強さ A/B (warmup 50 vs 400)

両アーム `--start shared_init.pt`・taskset12 固定・逐次実行。
- W50: `--warmup-steps 10` (=50ep), `--max-added-epochs 1000` → tag c3W50
- W400: `--warmup-steps 80` (=400ep), `--max-added-epochs 1500` → tag c3W400
- 共通: `--enemy-window 1 --self-play-ratio 0.5 --depth-skew 2.0 --search-turn-min 4 --search-turn-max 8
  --sims 64 --sim-concurrency 16 --train-num-games 64 --train-max-batch-size 512
  --train-trajectories-threshold 128 --train-minibatch-size 256 --epochs-per-step 5
  --peaks-per-rr 3 --finalists-target 3 --n-per-side 512 --num-games 256 --stage 3c --random --crit`
- 実時間: W50 ≈ 61分 (ep445 で finalist 到達), W400 ≈ 75分 (ep715 で finalist 到達)。

### 結果 (6 finalist 1プール総当たり, random+crit, n-per-side 512, num-games 256)

Bradley-Terry レート (平均0):

| 個体 | 手法 | レート | checkpoint |
|---|---|---:|---|
| c3W50#2 | c3W50 | **+4.0** | c3W50_ep445.pt |
| c3W400#2 | c3W400 | +2.9 | c3W400_ep715.pt |
| c3W50#1 | c3W50 | +1.2 | c3W50_ep315.pt |
| c3W400#1 | c3W400 | −1.9 | c3W400_ep600.pt |
| c3W400#0 | c3W400 | −2.0 | c3W400_ep450.pt |
| c3W50#0 | c3W50 | −4.2 | c3W50_ep175.pt |

手法平均: **c3W50 = +0.3 / c3W400 = −0.3**。

## 結論

- **warmup 50ep と 400ep で強さの差はほぼ無い** (手法平均差 0.6pt = ノイズ帯)。クロス対戦は全て
  48〜53% のほぼ互角。3c の対称・単純さゆえ個体差も 3b (K1b5 ±10.8) より小さい (±4)。
- 「自己対戦 warmup を長く焼くと強くなる」効果は、この 3c 設定では観測されず。
- **デプロイ個体 = c3W50_ep445.pt (+4.0, 個体最強)** を採用し Web (3c) へ。
