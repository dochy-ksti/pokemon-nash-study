## 目的

Phase 2 Stage 2b (最小比率、パルシェン B=166/D=106 ⇔ ヌメルゴン B=106/D=166、
ratio ≈ 1.57x) 決定論モードで、Stage 2a (2.74x) より狭い B/D 差でも AI が相手の
防御方向を識別して物理 (Crunch) と特殊 (Dark Pulse) を選び分けられるかを確認する。
Stage 2a-det 卒業時の設定 (`--num-games 8 --trajectories-threshold 8`) をそのまま
踏襲し、`--stage 2b` とチェックポイントパスのみ差し替えて再走する。

## コマンド

```bash
cd poke-ai3-python
uv run phase2-loop \
  --stage 2b \
  --num-games 8 \
  --chunk-threshold 16 \
  --trajectories-threshold 8 \
  --max-epochs 400 \
  --no-random --no-crit \
  --backend local \
  --checkpoint-path ../data/poke-ai3/phase2_stage2b_det/checkpoint.pt
```

- `--stage 2b`: 最小比率ステージ (B/D ratio 1.57x)
- `--no-random --no-crit`: 決定論モード (ダメージロール最大値固定、急所なし)
- `--num-games 8` / `--trajectories-threshold 8`: 8 ゲーム揃ったら 1 PPO update
- 相手 (P2) は Python 側で一様 `[0.5, 0.5]` を返すランダム相手

## 構成

Stage 2a-det と同一。

- モデル: Phase2PolicyValueModel (ModernBERT 隠れ 128 / 2 層 / max_pos=9)
- 観測スロット 8: my_hp, opp_hp, mask×2, my_species one-hot×2, opp_species one-hot×2
- PPO: lr=1e-4, γ=1.0, λ=0.95, clip=0.2, value_coef=0.5, entropy_coef=0.01,
  ppo_epochs=4, minibatch=64
- バトル: Lv50 gen9customgame、両者 Spe=90 同速、ターン上限なし、報酬±1/0
- 学習側固定 P1、相手 P2 は uniform random

## 結果

- 総 epoch: 400
- 初めて `correct_action_rate ≥ 0.95` を達成: epoch 72
- 連続 20 epoch `win_rate ≥ 0.95` 達成: epoch 132 (持続収束開始の目安)
- 末尾 100 epoch 平均:
  - win_rate = 0.985
  - correct_action_rate = 0.995
  - vs_cloyster_special_rate = 0.993 (パルシェンは Def 高 → 特殊が正解、高いほど良い)
  - vs_goodra_special_rate = 0.003 (ヌメルゴンは SpD 高 → 物理が正解、低いほど良い)
  - entropy = 0.021
  - raw_logits_std = 2.94
  - mean_return = 0.968

## 評価

- 卒業。ratio を 2.74x → 1.57x に狭めても soft-uniform 詰まりは再現せず、種族
  one-hot 観測でほぼ完璧に防御方向を識別 (correct_action_rate 0.995)。
- raw_logits_std ≈ 2.94 は Stage 2a-det (1.54) より大きく、より自信を持った policy。
  狭い比率でも logits が十分育っている。
- 収束は Stage 2a-det (連続 20 epoch 勝率達成 epoch 103) より少し遅い epoch 132
  だが、比率が狭く識別が難しいことを考えれば妥当な範囲。
- `vs_*_special_rate` の符号は Stage 2a-det と一致 (cloyster 高 / goodra ≈0)。
  Stage 2b の B/D 値が Stage 2a と逆向きにならず同じ向き (Cloyster=物理受け、
  Goodra=特殊受け) であることを確認。
