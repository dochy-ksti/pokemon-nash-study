## 目的

Stage 2b-rand (ratio 1.57x、確率モード) が soft-uniform で詰まった切り分けとして、
比率の広い Stage 2a (ratio 2.74x) を同じ `--random --crit` 条件で走らせ、比率幅が
ノイズ耐性を決めるかを確認する。

## コマンド

```bash
cd poke-ai3-python
uv run phase2-loop \
  --stage 2a \
  --num-games 8 \
  --chunk-threshold 16 \
  --trajectories-threshold 8 \
  --max-epochs 400 \
  --random --crit \
  --backend local \
  --checkpoint-path ../data/poke-ai3/phase2_stage2a_rand/checkpoint.pt
```

- `--random --crit`: 16 段ダメージロール + 急所を有効化 (確率モード)
- 他は Stage 2a-det / 2b-rand と同一

## 構成

Stage 2a-det と同一 (モデル・PPO ハイパラ・観測スロット)。差分は `--random --crit`。

## 結果 (卒業)

- 総 epoch: 400
- 初めて `correct_action_rate ≥ 0.95` を達成: epoch 119
- 連続 20 epoch `win_rate ≥ 0.95` 達成: epoch 147
- 末尾 100 epoch 平均:
  - win_rate = 0.990
  - correct_action_rate = 0.965
  - vs_cloyster_special_rate = 0.964 (パルシェン Def 高 → 特殊が正解、高いほど良い)
  - vs_goodra_special_rate = 0.033 (ヌメルゴン SpD 高 → 物理が正解、低いほど良い)
  - entropy = 0.124
  - raw_logits_std = 1.85
  - mean_return = 0.978

## 評価

- **卒業**。確率モード (急所 + 16 段ロール) でも、ratio 2.74x なら policy は
  防御方向を識別できた (correct 0.965)。
- 同じ確率条件で Stage 2b-rand (1.57x) が correct 0.506 / entropy 0.689 の
  soft-uniform に固着したのと対照的。**比率幅がノイズ耐性を決める**という
  仮説を支持する結果。
- 決定論版との比較:
  - 収束は遅い (det: sustain epoch 103 → rand: 147)
  - 最終的な自信もやや低い (det: logits_std 1.54 / 今回 rand: 1.85 は逆に高いが、
    entropy 0.124 は det 2a の 0.20 と同程度)
  - ※ Stage 2a-det は raw_logits_std 1.54 だったので、今回の rand 1.85 はむしろ
    十分に育っている。ノイズに対しても明確なシグナルが残ったと解釈できる。
- 含意: 確率モードへの移行は比率に依存する。狭い比率を確率モードで攻略するには
  (a) 決定論で卒業した重みからの fine-tune、(b) 報酬ノイズの抑制、(c) entropy_coef
  調整など、追加の工夫が要る。広い比率から段階的にノイズを入れるカリキュラムも候補。
