# stage3c 対称化と HP 5×5 unfold によるナッシュ均衡観察 (3b vs 3c)

- 日時: 2026-06-15 16:58 JST
- 担当ステージ: stage3b (非対称) / stage3c (対称化)

## 目的

stage3c は 3b のタイプ非対称(Cloyster vs GoodraHisui で SE 可否が偏る)を取り除き、
完全対称な対面に作り替えたシナリオ。狙いは「戦略を改善する」ことではなく、
**3b→3c でナッシュ均衡(混合戦略)がどう変化するかを観察する**こと。

検証したかった2つの仮説:

1. 3b にあった「Bulldoze が強く、ShockWave を撃たずに釣り交換するのが強い」という
   非対称由来の複雑さが、対称化で消え「一定確率で交換する単純なゲーム」になるか。
2. それでも 3HKO 由来の **HP 状況依存の混合戦略**は 3c でも残るか。

## 学習条件 (両ステージ共通)

- weak (nash_accumulation_weak, `--nash-weak`), hidden128, `--no-random --no-crit`
- sims64, search-turn 6..12, num-games 32, sim-concurrency 16, max-batch-size 320
- `--battle-seed 1` で seed 固定、`--snapshot-every 10`
- コマンド例:
  ```
  make train ARGS="--num-games 32 --sim-concurrency 16 --sims 64 \
    --search-turn-min 6 --search-turn-max 12 --no-random --no-crit \
    --stage 3c --nash-weak --hidden-size 128 \
    --battle-seed 1 --max-epochs 150 --max-batch-size 320 --snapshot-every 10 \
    --checkpoint-path data/poke-ai3/stage3c_weak_s1.pt"
  ```

## 収束判定 (eval-ckpt-vs-ckpt, 過去世代と head-to-head, 両方向合算 n=2048, SE≈0.011)

先に「先攻 P1 有利が無いこと」を自己対戦 4096 試合で確認 (P1 勝率 0.492 = 0.50±1σ)。
先攻決定は ChaCha8 の公平コインを毎ターン振り直し、クロスチーム対戦・先発網羅・
シードはゲーム/iter ごと別生成で対称。よって head-to-head の勝率はそのまま信頼できる。

### 3c (150ep まで)
最終世代 (ep150) の勝率:
- vs ep60 = 0.546 (+4.2σ), vs ep80 = 0.525 (+2.3σ)
- vs ep100 = 0.504, vs ep120 = 0.506, vs ep140 = 0.510 (いずれも互角)

→ **ep100 付近で均衡到達、以降フラットに収束。**

### 3b (150 → 250 → 500ep)
- ep150 最終は ep100 に +5σ で勝ち越し、まだ未収束だったため延長。
- 延長後の総当たり (ep150/180/200/220/250) で **ep200 が全世代に勝ち越す真のピーク**
  (平均勝率 .539)。ep220/250 は ep200 より弱く (.488)、ep200 以降は微劣化。
- さらに ep500 まで延長し ep200 をベンチに対戦:
  - ep200 vs ep300=0.523, ep350=0.520, ep400=0.495, ep450=0.513, ep500=0.522
  - → **ep500 までどの世代も ep200 を超えず**。劣化し続けるでも単調回復でもなく、
    ep200 をゆるいピークとして混合戦略均衡プラトー上を ±2σ で揺らぐだけ。

→ **3b は ep200 をピークに実質収束。** 非対称ゲームゆえ 3c より収束が遅く、
   世代間に非推移的(じゃんけん的)な揺らぎが出るのも混合戦略均衡の傍証。

採用 checkpoint: **3c = ep150 / 3b = ep200**。

## 結論

1. **仮説①: 確認。** 3b の中間的混合 (SW_v_Cl=0.46) と釣り交換 (BD_v_Cl=0.89) は
   タイプ非対称(Bulldoze 優位)が生む複雑さ。対称化した 3c では
   FS_v_Cl≈FP_v_Go(有利=0.16)/ FS_v_Go≈FP_v_Cl(不利=0.93)ときれいに2値化し、
   「有利なら居座る/不利なら引く」単純なゲームに。

2. **仮説②: 確認。** 3c でも HP 状況依存は残存。
   - FS_v_Cl(有利・居座り): 満タン×満タン 0.05 → 中削れ 0.19〜0.36 と交代増。
   - FS_v_Go(不利・交代): 自分HP高 0.93 → 自分<25% 0.41〜0.69 と居座りに転じる
     (引いても無駄な瀕死寸前)。
   → タイプ非対称の複雑さ(釣り交換)だけが消え、3HKO 由来の HP 依存は保持。

## 追記: 固定ルール AI との対戦 (eval-vs-rule) と「強さ」の取り違え

相手は決定論ルール AI (weakness 技を必ず撃つ)。各 n=512。

| checkpoint | vs rule 全体 | 先発有利(SE可) | 先発不利/等倍 |
|---|---|---|---|
| 3c ep150 | 0.588 (W301/L211) | 0.627 | 0.549 |
| 3b ep200 | 0.510 (W261/L251) | 0.558 | 0.460 (BD_v_Cl=0.437 負け越し) |
| 3b ep400 | **0.662** (W339/L173) | 0.678 | 0.646 (全対面で勝ち越し) |

注目: **3b ep200 と ep400 は head-to-head では完全に互角**
(再測 両方向合算 n=4096 で ep400 勝率 0.505 = +0.6σ。前回 0.495 と合わせ 5割確定)。
にもかかわらず **vs rule は ep400 (0.662) ≫ ep200 (0.510)**。

→ 同じ「均衡的強さ」の2点でも、決定論ルール相手の exploit 力は大きく異なる。
   混合戦略均衡の周りを彷徨ううち「対等だが固定相手への咎め方が違う」方策点を
   行き来している。**vs-rule 勝率でモデル選択すると均衡的な強さを取り違える**
   実例であり、収束判定を head-to-head (eval-ckpt-vs-ckpt) で行った判断の妥当性を裏付ける。

なお 3c (0.588) > 3b ep200 (0.510) は「3c が弱い学習結果」ではなく、3b は釣り交換
(BD_v_Cl) のような混合相手に効く戦略が決定論ルール相手だと裏目になりやすい、という
ゲーム構造差の反映。weak 学習はルール最大スポイルではなく混合ナッシュに寄せるため、
vs-rule 勝率が必ずしも伸びないのは想定どおり。
