# Stage3a タイプ弱点突き 学習実験

JST: 2026-06-05 15:23

## 目的

タイプ相性導入後の Stage3a (4技1v1, Cloyster vs Goodra-Hisui) で、AI が相手の種を
観測してタイプ弱点を突く技を選べるかを検証する。
- 対 Cloyster (みず/こおり): Shock Wave (でんき 2倍×低SpD) が正解。
- 対 Goodra-Hisui (はがね/ドラゴン): Bulldoze (じめん 2倍×低Def) が正解。
判定基準: 各弱点技の greedy 選択率 >90%。

## コマンド

```bash
cd poke-ai3-python
uv run train-loop --num-games 32 --sim-concurrency 16 --sims 64 \
  --search-turn-min 4 --search-turn-max 8 --no-random --no-crit --stage 3a \
  --max-epochs 40 --chunk-threshold 512 \
  --checkpoint-path ../data/poke-ai3/stage3a.pt
```

- 決定論モード (--no-random --no-crit)。両陣営ランダムに Cloyster/Goodra を先発、4技共通。
- 評価指標は diagnostics.py を合法手マスクから弱点技を判定する方式に更新済み
  (Stage2a/3a 共通)。ログ: `data/poke-ai3/stage3a_train.log`。

## 結果

| epoch 帯 | correct_action_rate | vs_cloyster_weakness_rate | vs_goodra_weakness_rate |
|---|---|---|---|
| 1–10  | ~0.5 (振動) | 0/1 で全振れ | 1/0 で全振れ |
| 11–26 | **1.0000** | **1.0000** | **1.0000** |
| 27–40 | 0.83–0.97 | ~1.0 | 0.65–0.94 |

- **中盤 (epoch 11–26) で correct_action=1.0・両弱点率=1.0 に到達。判定基準 (>90%) を満たし、
  弱点突きの学習に成功**。
- 初期 (1–10) はポリシーが単一手に潰れ、相手種別ごと丸ごと正/誤になる典型的な collapse。
- 後半 (27–40) に対 Goodra 側で弱点率が 0.65–0.94 に揺れる (過学習的なぶれ)。実用上は
  early stopping で epoch 11–26 帯を採用すれば十分。

## 所見・フォローアップ

- win_rate が後半 0.0–0.18 まで低下 (自己対戦の鏡像なら 0.5 付近が期待値)。弱点突き学習
  自体には影響しないが、win_rate 算出 / value 学習の癖の可能性。要確認。
- 後半の弱点率のぶれは、エントロピー/学習率や early stopping で抑えられる見込み。
