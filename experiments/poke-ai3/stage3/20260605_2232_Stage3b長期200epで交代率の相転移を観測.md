# Stage3b 長期200epで交代率の相転移を観測

JST: 2026-06-05 22:32

## 目的

前検証(`20260605_2134`)は5〜20epのみで「不利対面の交代確率 ~25% が混合均衡」と結論
づけていた。これに懐疑があり、**epoch を大幅に延ばして交代戦略の時間発展を観察**する。

## コマンド

```bash
cd poke-ai3-python
uv run train-loop --num-games 32 --sim-concurrency 16 --sims 64 \
  --search-turn-min 6 --search-turn-max 12 --no-random --no-crit --stage 3b \
  --max-epochs 200 --chunk-threshold 512 \
  --checkpoint-path ../data/poke-ai3/stage3b_long.pt
```

前回検証と sims/探索深/num-games/W/chunk を完全に揃え、epoch のみ 20→200 へ拡大
(スクラッチ別ckpt)。exit 0 で200ep完走、workspace 影響なし。

## 結果 — 不利対面の交代確率 (model/teacher, 区間平均)

| epoch区間 | model_switch_prob | teacher_switch_prob |
|---|---|---|
| 1–10 | 0.335 | 0.346 |
| 11–30 | **0.252** | **0.264** | ← 前回が「混合均衡」と誤読した谷 |
| 31–50 | 0.571 | 0.573 | ← 相転移 |
| 51–100 | 0.620 | 0.622 |
| 101–150 | 0.678 | 0.678 |
| 151–200 | **0.726** | **0.725** |

win_rate は全区間 ~0.5(対称自己対戦として正常)。model は全epochで teacher をほぼ
完全追従(縦切りの配線健全性は前回どおり)。

## 結論

1. **前回の「~25%混合が均衡」は誤り。** 5〜20ep は交代率が一時的に下がる**谷**
   (~0.26)で、ちょうどそこを切り取って均衡と誤読していた。
2. **epoch 30→40 付近で相転移**が起き、不利対面の交代確率が ~0.26 → ~0.57 へジャンプ。
   以降も単調上昇し、151–200ep では **~0.73** に達する。打ち切りの兆候はなく、さらに
   epoch を伸ばせばもう少し上がる可能性がある。
3. 前回残課題1の切り分け(1a 真の混合均衡 / 1b lookahead が浅く過小評価)は **1b 寄り**
   と判明: sims/探索深を変えずとも、**学習が進み両者の play が洗練されるほど教師自身の
   交代評価が上がる**。交代率は学習の進行に強く依存する量で、短期 epoch のスナップ
   ショットで「均衡値」を語ってはいけない。
4. 真の自己対戦均衡は「不利対面で攻撃優位(~25%交代)」ではなく、**過半数(70%+)で
   交代する**戦略。grill-me 質問4 の switch ループ懸念は、均衡を交代過小には導かなかった。

## 追記 — 継続学習 +100ep で漸近上限を確認 (JST 23:17)

200ep到達 ckpt (`stage3b_long.pt`) から optimizer ごと継続し、同条件で +100ep。

```bash
uv run train-loop --num-games 32 --sim-concurrency 16 --sims 64 \
  --search-turn-min 6 --search-turn-max 12 --no-random --no-crit --stage 3b \
  --max-epochs 100 --chunk-threshold 512 \
  --checkpoint-path ../data/poke-ai3/stage3b_long.pt
```

| 継続区間 | model_switch_prob | teacher_switch_prob | win_rate |
|---|---|---|---|
| 1–20 | 0.720 | 0.718 | 0.46 |
| 21–40 | 0.741 | 0.742 | 0.48 |
| 41–60 | 0.757 | 0.756 | 0.50 |
| 61–80 | 0.742 | 0.741 | 0.51 |
| 81–100 | 0.755 | 0.757 | 0.50 |

- 200ep末の ~0.73 から **~0.75 で横ばい(プラトー)**。前半200epの急上昇(谷0.26→0.73)
  に対し伸びは +0.02 程度で頭打ち。**交代率の漸近上限は ~0.75 前後**と確定。
- 「不利対面では3/4以上で交代」が Stage3b の自己対戦均衡。model は引き続き teacher を
  完全追従、win_rate ~0.5 維持。
- 合計300ep 到達 ckpt は `data/poke-ai3/stage3b_long.pt` に上書き保存。
